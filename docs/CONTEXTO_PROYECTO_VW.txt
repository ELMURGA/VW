# CONTEXTO COMPLETO - PROYECTO WEB VOLKSWAGEN SEVILLA UTREWAL

## DESCRIPCIÓN GENERAL DEL PROYECTO

Se ha creado un sitio web completo y moderno para el concesionario oficial Volkswagen Sevilla (Utrewal) con dos ubicaciones: Utrera y Alcalá de Guadaíra. El sitio está diseñado con un enfoque premium, utilizando diseño moderno con glassmorphism, gradientes, animaciones suaves y diseño responsive.

---

## TECNOLOGÍAS UTILIZADAS

- **HTML5** semántico
- **CSS3** con variables CSS (custom properties)
- **JavaScript** vanilla (sin frameworks)
- **Google Fonts**: Inter (300, 400, 500, 600, 700, 800)
- **Font Awesome 6.5.1** para iconos
- **Diseño responsive** para móvil, tablet y desktop

## FUNCIONALIDADES JAVASCRIPT (script.js)

1. **Header scroll**: Se hace más compacto al hacer scroll
2. **Menú móvil**: Toggle hamburger menu
3. **Slider hero**: Autoplay con indicadores y navegación
4. **Filtros de vehículos**: Por categoría
5. **Scroll to top**: Botón flotante
6. **Formularios**: Validación y simulación de envío
7. **Widget de accesibilidad**:
   - Cambiar tamaño de fuente
   - Alto contraste
   - Reducir animaciones
8. **Banner de cookies**: Con localStorage para recordar preferencia
9. **Smooth scroll**: Para enlaces internos
10. **Animaciones on scroll**: Elementos aparecen al hacer scroll

## CARACTERÍSTICAS DE DISEÑO

### Efectos visuales:
- Glassmorphism (backdrop-filter: blur)
- Gradientes suaves
- Animaciones CSS (hover, transitions)
- Sombras con niveles (shadow-sm, shadow-md, shadow-lg, shadow-xl)
- Border radius consistentes

### Responsive breakpoints:
- Mobile: hasta 768px
- Tablet: 768px - 1024px
- Desktop: 1024px+

### Tipografía:
- Fuente: Inter
- Tamaños: desde 0.75rem hasta 3rem
- Pesos: 300 (light) a 800 (extra-bold)
