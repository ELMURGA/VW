Declaración de Accesibilidad

Compromiso con la Accesibilidad

En Utrewal S.A. nos comprometemos a garantizar la accesibilidad digital para todas las personas, incluyendo aquellas con discapacidad.

Situación de cumplimiento

Utrewal S.A. se ha comprometido a hacer accesible su sitio web de conformidad con el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público, que transpone la Directiva (UE) 2016/2102 del Parlamento Europeo y del Consejo.

Este sitio web es parcialmente conforme con el RD 1112/2018 debido a la falta de conformidad de algunos aspectos que se indican a continuación.

Contenido no accesible

El contenido que se recoge a continuación no es accesible por los siguientes motivos:

  • Pueden existir algunos contenidos de terceros que no están completamente accesibles.
  • Es posible que algunos documentos PDF no cumplan todos los requisitos de accesibilidad.
  • Algunos vídeos pueden carecer de subtítulos o audiodescripción.

Características de accesibilidad de este sitio

Tamaño de texto ajustable

Puede aumentar o reducir el tamaño del texto según sus necesidades.

Alto contraste

Opción de visualización en alto contraste para mejorar la legibilidad.

Navegación por teclado

Todos los elementos interactivos son accesibles mediante teclado.

Reducir animaciones

Posibilidad de desactivar las animaciones para usuarios sensibles.

Textos alternativos

Las imágenes incluyen descripciones alternativas para lectores de pantalla.

Estructura semántica

El sitio utiliza HTML semántico para una mejor navegación.

Widget de Accesibilidad

Nuestro sitio web incluye un widget de accesibilidad (icono en la parte inferior derecha de la pantalla) que permite:

  • Aumentar/Reducir el tamaño del texto para facilitar la lectura
  • Activar el modo de alto contraste para mejorar la visibilidad
  • Reducir las animaciones para usuarios con sensibilidad al movimiento

Estas preferencias se guardan automáticamente y se mantienen durante su navegación.

Preparación de la presente declaración

Esta declaración fue preparada en diciembre de 2024. El método empleado para preparar la declaración ha sido una autoevaluación llevada a cabo por el propio organismo.

Observaciones y datos de contacto

Puede realizar comunicaciones sobre requisitos de accesibilidad (artículo 10.2.a del RD 1112/2018) como por ejemplo:

  • Informar sobre cualquier posible incumplimiento por parte de este sitio web
  • Transmitir otras dificultades de acceso al contenido
  • Formular cualquier otra consulta o sugerencia de mejora relativa a la accesibilidad del sitio web

Procedimiento de aplicación

Si una vez realizada una solicitud de información accesible o queja, ésta hubiera sido desestimada, no estuviera de acuerdo con la decisión adoptada, o la respuesta no cumpliera los requisitos contemplados en el artículo 12.5 del RD 1112/2018, la persona interesada podrá iniciar una reclamación.

Igualmente se podrá iniciar una reclamación en el caso de que haya transcurrido el plazo de veinte días hábiles sin haber obtenido respuesta.

Contenido opcional

Este sitio web aplica los requisitos de la Norma UNE-EN 301549:2019, nivel de accesibilidad AA de las WCAG 2.1, siguiendo las recomendaciones del World Wide Web Consortium (W3C).
